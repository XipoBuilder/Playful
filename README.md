## 🚀 快速开始

- **在线体验**：点击上方表格中的“在线玩”链接，无需安装。
- **本地运行**：克隆本仓库，双击任意游戏子文件夹中的 `index.html`。

```bash
git clone https://github.com/XipoBuilder/Playful.git
cd Playful/gomoku
open index.html   # 或双击
```
